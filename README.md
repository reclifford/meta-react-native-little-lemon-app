Final project for Meta's React Native certification course on Coursea.

Includes the following features:

-Welcome screen.
-Subscribe screen.
-Menu screen.
-React Navigation Library.

Functionality:
User clicks the 'subscribe' button on Welcome screen to navigate to the Subscribe screen.
User enters a valid email address (validated as inputed) to be able to access the 'subscribe' button.
If user entered valid email, user will recieve a confirmation alert that will then redirect them back to Welcome screen.
